From courses to program outcomes

Program outcomes (PO) definition

Each program, pp, is defined as a set of program outcomes, OpO_p.

Course activities to learning outcomes (LO)

Let CpC_p be the set of core courses in a diploma programme, pp. For any course, c∈Cpc \in C_p, there is a set of learning outcomes, LcL_c.

Each course has a set of assessment activities, AcA_c. Each activity of a course, a∈Aca \in A_c, contributes to one or more of the course learning outcomes. For example, LOcontribc,a,lLOcontrib_{c,a,l} is the contribution of activity a∈Aca \in A_c to learning outcome l∈Lcl \in L_c for course cc. These contributions form a preliminarly A-to-LO contribution matrix for the course, X′cX'_c , whose elements are x′a,lc=LOcontribc,a,lx'^{c}_{a,l}= LOcontrib_{c,a,l} .

For the sake of simplicity the matrix X′cX'_c is a binary matrix, i.e. the values LOcontribc,a,lLOcontrib_{c,a,l} can take only values 0 or 1. However, there is also the problem that activities are of different weightage, wc,aw_{c,a}, forming a weightage matrix of the course activities, 𝐰𝐜\mathbf{w_c}. For example, weight of final exam can be 30% (i.e. 0.3) whereas a homework can be 10%. To account for these differences we use a weighted A-to-LO contribution matrix: Xc=𝐰𝐜X′cX_c=\mathbf{w_c} X'_c

Each course also has an ECTS credit value, ectscects_c which will be used later for the level of contribution.

From LO to PO

In order to assess achievement of program outcomes, OpO_p, for a course in the program, c∈Cpc \in C_p, its learning outcomes will be mapped to program outcomes. For example contribution value of a learning outcome, l∈Lcl \in L_c to a program outcome, o∈Opo \in O_p is denoted as POcontribc,l,oPOcontrib_{c,l,o}. These contributions form a LO-to-PO contribution matrix for the course, YcY_c, whose elements are yl,oc=POcontribc,l,oy^c_{l,o}=POcontrib_{c,l,o}.

For the sake of simplicity the matrix YcY_c is a binary matrix, i.e. the values POcontribc,l,oPOcontrib_{c,l,o} can take only values 0 or 1.

Evaluation based on course designs: New scheme (for the future)

Here the aim is to evaluate the level which courses in the curriculum, via their learning outcomes, support the program outcomes.

A first step to compute this would be to compute contribution of a course to programme outcomes as Tc=YcTXcT^c=Y_c^T X_c, and collapse the activities level to obtain a vector of PO contributions of a course. Let us denote this resulting vector as 𝐳𝐜\mathbf{z^c} whose elements represent contribution of all activities of the course to each program outcome oio_i, i.e. zic=∑jti,jz^c_i=\sum_{j}t_{i,j}

One can then sum this over all courses in a program’s curriculum to find the support for program outcomes: ∑c∈Cp𝐳𝐜\sum_{c \in C_p}\mathbf{z^c}. However, the main problem here is that courses have different ECTS values, different number of learning outcomes and different number of program outcomes each LO contributes to. Thus, values in the total contribution vector of a course to program outcomes, TotalContribc=∑𝐳𝐜TotalContrib_c=\sum\mathbf{z^c}, can be very different across courses and not necessarily proportional to course’s ECTS credits. Hence a normalization is needed. A straightforward solution is to ensure that sum of contributions of a course is equal to its ECTS credit, by normalizing as:

𝐳𝐧𝐨𝐫𝐦𝐜=𝐳𝐜ectscTotalContribc \mathbf{z^c_{norm}}=\mathbf{z^c} \frac{ects_c}{TotalContrib_c}

To find the support of the whole curriculum for each of the program outcomes, one must sum normalized contributions of all courses as follows (please see the practical note in the next section for a tentative, realistic calculation of the following):

∑c∈Cp𝐳𝐧𝐨𝐫𝐦𝐜\sum_{c \in C_p} \mathbf{z^c_{norm}}

A practical evaluation based on existing information

In IEU system the syllabus declares the 𝐳𝐜\mathbf{z^c} matrix regardless of the activities’ contributions. Ideally the declared matrix and matrix computed as a bove should be the same; i.e. in the future we must use the computed contributions matrix, POcontribPOcontrib, instead of directly entering some numbers into the table in the syllabus. For now, however, we must skip the computations and use the declared contributions in the syllabus. Thus, for the rest of this section, assume that 𝐳𝐜\mathbf{z^c} is obtained from syllabus, rather than being derived by following A-to-LO and LO-to-PO contribution levels.
